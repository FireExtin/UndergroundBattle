## 对“现在试跑原型”的含义

- 现在的原型定位是 **“基础包规则核的最小可运行沙箱”**，不是“可完整打一局基础包实卡”。
- 你现在能稳定体验到的：
  - 动作提交、优先权与堆叠变化
  - 攻击/调查/伤害/销毁/得分/终局
  - 护盾 V1 拦截（`shield_consumed`）
  - marker/face-down 的状态与投影变化
- 你暂时看不到的（还没接）：
  - 完整隐秘对抗
  - 大多数地区赢取触发与任务持续效果
  - 核心区与大规模基础包卡义

## 原型试跑（最短路径）

1. 启动后端：

```bash
go run ./server/cmd/api
```

2. 启动前端：

```bash
cd web
npm run dev
```

3. 打开：

- `http://localhost:5173`

建议先在 Live Debugger 连续测试：
- `Queue Operation`（含目标）
- `Declare Attack`
- `Declare Investigation`
- `Set/Remove Marker`
- `Set Face-Down`
- `Reset Sandbox`
